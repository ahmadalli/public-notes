## Fun Projects

### rtl-433

`rtl_433` is a command-line program that decodes signals from many low-cost wireless sensors and devices using an RTL-SDR receiver.

These are some interesting projects that use `rtl_433`:

- [TPMSbash](https://r-c-y.net/posts/tpmsbash/): Read TPMS (Tire Pressure Monitoring System) data from your car.
- [rtl_433-hass-addons](https://github.com/pbkhrv/rtl_433-hass-addons): Home assistant integration for `rtl_433`.
