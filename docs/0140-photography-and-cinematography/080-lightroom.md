# Adobe Lightroom

## Edit Versioning

I use the following pattern to name my edit versions: `<Color/Monochrome> - <Mood/Tone, e.g. Saturated> - <Target, e.g. Web> - <version number, e.g. v1>`

## Catalogue Issues in Classic

### Catalogue Size

- Denoise data is stored in XMP and the catalogue file. If you have a large number of photos with denoise applied, the catalogue file can grow significantly in size.
