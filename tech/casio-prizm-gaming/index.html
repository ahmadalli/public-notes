Gaming on Casio Prizm

I had a Casio fx-CG50 calculator nearby for a short time and looked into the games that can be played on it:

  • Game Boy Emulator: tswilliamson/prizoop emulates Game Boy games.
    • I found some Game Boy games on emulatorgames.net
  • There are other games on cemetech.net but not all of them work. For example I couldn't get Flappy Bird to work, but I didn't try looked into it much.
